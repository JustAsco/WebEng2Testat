# WE2 Testat template

The template contains a basic setup of React + TypeScript + Vite + React Router

## Usage

Open your terminal and run `npm i` to install dependencies and then `npm run dev`to start your dev environment. Click the link in your terminal to view the live preview.

### Other Commands

In the package.json, you can find preconfigured commands. Run them with `npm run <command>`.


## Prettier

This template is configured for [Prettier](https://github.com/prettier/prettier). Find the docs under [prettier.io](https://prettier.io/docs). Install the [Prettier VS Code Extension](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)

## The root directory

- **`README.md`** – Documentation file that typically includes setup instructions, usage details, and developer notes.
- **`index.html`** – The main HTML template used by Vite to load and bootstrap the React application.
- **`.gitignore`** – Specifies which files and folders Git should ignore (e.g., build output, dependencies, environment files).
- **`package.json`** – Defines project metadata, npm scripts, dependencies, and other project configuration.
- **`package-lock.json`** – Auto‑generated file that locks exact dependency versions to ensure reproducible installs.
- Other Config files
  - **`.prettierrc`** – Contains config for Prettier.
  - **`.prettierignore`** – Defines what Prettier should ignore.
  - **`eslint.config.js`** – Contains ESLint configuration for enforcing consistent code style and catching errors.
  - **`tsconfig.app.json`** – TypeScript configuration specifically for the application source files, usually extending the base config.
  - **`tsconfig.json`** – The base TypeScript configuration that defines general compiler options and project settings.
  - **`tsconfig.node.json`** – TypeScript configuration for Node‑based files such as Vite config or build scripts.
  - **`vite.config.ts`** – Configuration for Vite, including plugins, dev server options, and build settings.

### Folders

- **`public/`** – Static assets served as‑is, e.g. images; files here are not processed by Vite.
- **`src/`** – The main source directory containing React components, styles, TypeScript code, and application logic.
- **`node_modules/`** – A Folder generated by NPM containing installed dependencies for the project.
